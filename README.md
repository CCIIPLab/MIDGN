# MIDGN

## Code

This is the source code for AAAI 2022 Paper: Multi-View intent disentangle graph networks for bundle recommendation.

## Requirements

- Python 3
- PyTorch = 1.8.0
- torchvision = 0.9.0
- torchaudio = 0.8.0
- torch-scatter = 2.0.7
- torch-sparse = 0.6.10
- tensorboardX = 2.3
- tqdm

## Usage


Train and evaluate the model:
~~~~
python main.py
~~~~
